# AI Assisted AML SAR Automation System

## Overview
This project focuses on automating the generation of Suspicious Activity Reports using AI and structured workflows.

## Problem
Manual SAR generation is time consuming, repetitive and often inconsistent, making it difficult to maintain efficiency and accuracy.

## Approach
Exploring how AI can assist in generating structured SAR narratives by using logical workflows and prompt based outputs.

## Work Done
Researched the SAR generation process  
Studied how automation can improve efficiency  
Designed basic workflow for system understanding  

## Key Focus
Reducing manual effort  
Improving consistency  
Understanding real world financial systems
